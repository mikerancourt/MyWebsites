Introduction to HTML

HTML=HyperText Markup Language. Describes the structure of a web page. Uses tags. Most elements are enclosed in angled brackets called tags, like this: <html>

To comment html code, start with "!-- " and end with "-- ", like this:

<!-- COMMENT GOES HERE -->

<main> is an element that gives more descriptive structure to your HTML, makes it easier to read, and helps with SEO (Search Engine Optimization)

Images

Start with <img> tag. Self-closing. Inside, add src="image url " and then add alt=”this is alt text in case image breaks, old browser, or screen readers”

EXAMPLE: <img src=”url” alt=”alt text here.”>

Anchors

<a></a> is an anchor. It links to content outside your page. Anchors need a destination web address called an "href " attribute. They also need anchor text.

Example: <a href=”https://freecodecamp.org"> this links to freecodecamp.org </a>

Your browser displays “this links to freecodecamp.org” and your user can click the text.

Anchors can also link internally. In the href, just add # and id name.

Example: you have an h2 with this info: <h2 id=”contacts-header”>Contacts</h2>.

Now you can create an unordered list or navbar, and you can call that same id with the number sign #.

Href has an attribute called target. Set it to “_blank” so the link opens in a new browser window. Example:

Click this link to freecodecamp.org for you to follow.
